# Command syntax #

{% apibody %}
string.downcase() &rarr; string
{% endapibody %}

# Description #

Downcases a string.

__Example:__

```py
r.expr("Sentence about LaTeX.").downcase().run(conn)
```

Result:

```py
"sentence about latex."
```
