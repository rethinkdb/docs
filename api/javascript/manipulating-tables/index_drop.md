# Command syntax #

{% apibody %}
table.indexDrop(indexName) &rarr; object
{% endapibody %}

# Description #

Delete a previously created secondary index of this table.

__Example:__ Drop a secondary index named 'code_name'.

```javascript
r.table('dc').indexDrop('code_name').run(conn, callback)
```
