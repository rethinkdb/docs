# Command syntax #

{% apibody %}
table.indexList() &rarr; array
{% endapibody %}

# Description #

List all the secondary indexes of this table.

__Example:__ List the available secondary indexes for this table.

```javascript
r.table('marvel').indexList().run(conn, callback)
```
