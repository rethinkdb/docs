# Command syntax #

{% apibody %}
r.json(json_string) &rarr; value
{% endapibody %}

# Description #

Parse a JSON string on the server.

__Example:__ Send an array to the server.

```javascript
r.json("[1,2,3]").run(conn, callback)
```
